# Rust Desktop v1.0 - Release Notes

## Package Information

- **Version**: 1.0
- **Platform**: Windows 10/11 (64-bit)
- **Package Size**: ~87 MB (compressed)
- **Format**: Self-contained executable (no .NET installation required)

## What's Included

✅ **RustDesktop.exe** - Main application (single-file executable)
✅ **738 Rust item icons** - All icons included in Icons folder
✅ **C4 application icon** - Used for taskbar, window, and system tray
✅ **README.txt** - User-friendly installation and usage guide
✅ **LICENSE** - MIT License
✅ **All dependencies** - Fully self-contained, no additional installs needed

## Features

- 🗺️ Interactive map with vending machine locations
- 🛒 Vending machine search and filtering
- 🚨 Raid alerts with desktop notifications
- 🌍 World event notifications (cargo ships, helicopters, etc.)
- 📊 Real-time data updates
- 🔔 System tray integration with C4 icon
- 🔊 Sound notifications for alerts

## Distribution

The package is ready for distribution:

**Location**: `release/RustDesktop-v1.0-win-x64.zip`

**To distribute:**
1. Share the ZIP file with users
2. Users extract and run `RustDesktop.exe`
3. No installation required - fully portable

## System Requirements

- Windows 10/11 (64-bit)
- Steam installed and logged in
- Rust game installed
- Node.js (optional, for FCM pairing)

## Installation Instructions for Users

1. Extract the ZIP file to any folder
2. Run `RustDesktop.exe`
3. Follow the on-screen setup:
   - Connect Steam
   - Pair Server (requires Node.js or mobile app pairing)
   - Connect to Rust+

## Notes

- The application is fully portable - can run from any location
- All Rust item icons are included (738 icons)
- C4 icon appears in taskbar, window title bar, and system tray
- Desktop notifications appear in bottom-right corner
- App minimizes to system tray (right-click to exit)
